WEBGL2 · ZERO DEPENDENCY

Viscous Fluid Demos

化粧品サイト向けの「原液・美容液」質感に特化したWebGL2流体エフェクトライブラリ。
依存ゼロ・script タグ1行で埋め込み可能。デモページを以下にまとめています。

MAIN — FluidSim (fluid-sim.js)
STUDIO

スタジオ(調整UI付き)

全パラメータをリアルタイム調整できる統合デモ。プリセット切り替え、背景切り替え、LPオーバーレイ表示にも対応。
PRESETS

スタジオ — プリセット直リンク

スタジオ画面をUIなしで、プリセットを指定して直接開くリンク集です。

原液(黒) 原液(白) クリア スプラッシュ
EMBED

埋め込みサンプル(LP実装例)

実際のLPヒーローセクションを想定した埋め込みパターン。セクション内埋め込み・オーバーレイ構成の参考に。

 DEMO

原液デモ(ダーク背景)

黒背景での原液系グロス表現の単体デモ。?ui=1で調整パネルを表示。

 DEMO

原液デモ(ホワイト背景)

白背景プロファイルと原液/クリアジェルのプリセット切替に対応したデモ。

 DEMO

スプラッシュデモ(低粘性・カラフル)

vorticity confinementとレインボーカラーモードを使った、低粘性で弾けるような表現。
EFFECTS — スタンドアロンエフェクト集
EFFECT

金粉ラメシマー

GPUパーティクルの金粉フレークが揺らめきながら煌めく単体エフェクト。?ui=1で調整パネルを表示。

 EFFECT

薄膜干渉パール

シャボン膜のような虹色パールの薄膜干渉を、ポインターの波紋とともに表現。?ui=1で調整パネルを表示。

 EFFECT

水面コースティクス

水面越しに揺れる光のコースティクス網目と、その場で広がる波紋リング。?ui=1で調整パネルを表示。

 EFFECT

シルクサテン

異方性反射が布目に沿って流れるサテン地の質感。ポインターで軽く押し込める。?ui=1で調整パネルを表示。
BONUS — ViscousFluid (viscous-fluid.js)
LEGACY

メタボール版(軽量・旧実装)

格子流体ではなくメタボールフィールドで質感を表現する、軽量な旧実装。ボーナス収録。
